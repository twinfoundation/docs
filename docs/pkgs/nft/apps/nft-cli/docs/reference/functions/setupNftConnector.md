# Function: setupNftConnector()

> **setupNftConnector**(`options`): `INftConnector`

Setup the NFT connector for use in the CLI commands.

## Parameters

### options

The options for the NFT connector.

#### nodeEndpoint

`string`

The node endpoint.

#### network?

`string`

The network.

#### vaultSeedId?

`string`

The vault seed ID.

#### walletAddressIndex?

`number`

The wallet address index.

## Returns

`INftConnector`

The NFT connector.
