# Function: getDeploymentMnemonic()

> **getDeploymentMnemonic**(`network`): `Promise`\<`string`\>

Get the deployment mnemonic for a network.

## Parameters

### network

`NetworkTypes`

The target network.

## Returns

`Promise`\<`string`\>

The mnemonic string.

## Throws

GeneralError if mnemonic is not found or invalid.
