# Enumeration: ErrorCorrectLevel

Error correction level to use for the QR Code.
Based on https://github.com/kazuhikoarase/qrcode-generator/ .

## Enumeration Members

### H

• **H** = ``2``

30% Error correction.

___

### L

• **L** = ``1``

7% Error correction.

___

### M

• **M** = ``0``

15% Error correction.

___

### Q

• **Q** = ``3``

25% Error correction.
